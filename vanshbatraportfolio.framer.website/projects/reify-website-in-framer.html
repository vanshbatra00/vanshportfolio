Smart Home Automation

IoT
Automation
SMS Control

It is one of my IoT-based projects which has control on home appliances by Wi-Fi as well as SMS.

My Approach

I developed a smart home automation system designed to control all home appliances without being visibly intrusive. The system is powered by the ESP32 module, which provides seamless Wi-Fi connectivity. Additionally, I integrated the SIM900L GSM module, enabling remote control through SMS, ensuring control even in areas with unreliable Wi-Fi.

For the UI, I implemented the Blynk IoT platform, which offers an intuitive and real-time interface for monitoring and managing the appliances. The system communicates with the Blynk server over MQTT, ensuring low latency and efficient data transfer.

Vision and Innovation

While developing this project, my primary focus was to address the issue of unreliable internet connectivity. To achieve this, I implemented an innovative SMS control mechanism alongside Wi-Fi, allowing the system to switch between the two based on network availability. This failover mechanism ensures continuous operability, making the system highly adaptable for remote areas with inconsistent or no internet access.

Achievements
